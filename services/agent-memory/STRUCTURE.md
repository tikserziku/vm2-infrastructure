# Agent Memory System v1.0

## Структура
- /knowledge/ - База знаний (что где лежит)
- /history/ - История операций
- /embeddings/ - Векторные индексы для RAG

## Файлы
- vm_map.json - Карта файловой системы VM
- services.json - Все сервисы и их конфиги
- secrets.json - Где лежат секреты (зашифрованные пути)
- projects.json - Проекты и их расположение
- changelog.log - История изменений
